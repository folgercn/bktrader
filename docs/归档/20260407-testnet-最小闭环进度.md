### 5. 为什么需要 `strategyVersionId`
对 `LIVE` 账户执行 `POST /api/v1/orders` 时，系统要求：
- 要么提供 `strategyVersionId`
- 要么账户已经挂了 linked runtime session

否则会报错：
- `live order requires strategyVersionId or a linked runtime session`

为了做最小烟雾测试，本次采用：
- 提供现有策略版本 ID
- 同时加 `metadata.manualTest=true`

这样可以跳过 runtime readiness 检查，先验证真实交易链路。
